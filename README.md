# Timetable App

This is a full-stack timetable management application with:

- 🖥️ Frontend: React (Vite)
- ⚙️ Backend: Node.js + Express

## 📂 Folder Structure

- `/client` → React frontend
- `/server` → Express backend

## 🛠️ Getting Started

### Frontend
```bash
cd client
npm install
